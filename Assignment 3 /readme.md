# Handwritten Digit Generation with DCGAN


## Objective:
#### In this assignment, you will implement a Deep Convolutional Generative Adversarial Network (DCGAN) to generate realistic handwritten digits using the MNIST dataset. The goal is to understand how GANs work, practice building generator and discriminator networks, and evaluate the quality of generated samples.

### Learning Objectives: By completing this assignment, you will:
● Understand the architecture of GANs (Generator vs. Discriminator).

● Learn how adversarial training works.

● Gain hands-on experience with PyTorch/TensorFlow for deep learning.

● Explore how neural networks can generate synthetic data


### Dataset:
● MNIST dataset (handwritten digits, 28×28 grayscale images).

● Freely available via PyTorch (torchvision.datasets.MNIST) or TensorFlow (tf.keras.datasets.mnist).

● No license restrictions — safe for academic use.


### Deliverables:
● Code implementation (Python, PyTorch or TensorFlow).

● Generated sample images (at least 10 digits).


### Constraints:
● The assignment is designed to run on modest hardware (single GPU with 4–8 GB VRAM or Google Colab).

● Training should complete within 1–2 hours on such hardware.

● Use lightweight models (avoid very deep networks).
